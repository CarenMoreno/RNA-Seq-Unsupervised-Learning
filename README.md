# RNA-Seq-Unsupervised-Learning
Unsupervised learning analysis of RNA-Seq cancer data using PCA, t-SNE, UMAP and MDS

This project explores gene expression data from cancer samples using unsupervised learning techniques.

## Methods used
- PCA (Principal Component Analysis)
- t-SNE
- UMAP
- MDS

## Objective
To identify patterns and clustering behavior in high-dimensional RNA-seq data.

## Dataset
- Gene expression data (RNA-seq)
- Cancer type labels

## Results
UMAP and t-SNE showed the best cluster separation, revealing biologically meaningful groupings between cancer types.
